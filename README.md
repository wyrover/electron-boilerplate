# electron-boilerplate

# 安装

```
cnpm install jquery --save
cnpm install handlebars handlebars-layouts mkdirp --save-dev
cnpm install electron-prebuilt --save-dev
```

# 编译运行

``` sh
npm run build
```

# 运行

``` sh
npm run start
```
